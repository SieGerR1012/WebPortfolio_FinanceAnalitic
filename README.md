# WebPortfolio_FinanceAnalitic

Персональный многостраничный сайт-портфолио Финансового аналитика с адаптивной вёрсткой и простой анимацией.

## Возможности

- Адаптивный дизайн (мобилки, планшеты, десктоп)
- Анимации при скролле (ScrollReveal)
- Секции: Главная, Обо мне, Образование, Услуги, Проекты, Контакты

## Технологии

- HTML5
- CSS3
- JavaScript
- [BoxIcons](https://boxicons.com/)
- [ScrollReveal](https://scrollrevealjs.org/)
